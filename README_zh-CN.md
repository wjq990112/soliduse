## 描述

一个 SolidJS 的工具函数库（受 [ahooks](https://ahooks.js.org/zh-CN/) 和 [VueUse](https://vueuse.org/) 启发）。

## 特性

- 💯 高质量：每个函数都有完整的测试用例且覆盖率超过 80%。
- 🚀 SSR 支持：每个函数都同时支持 CSR 和 SSR 且行为一致。
